### 2️⃣ 판매상품 리스트 페이지 (`/products`)

**확인 사항**:
- [ ] 카테고리 필터 버튼 5개 (전체, 아우터, 이너웨어, 액세서리, 신발)
- [ ] 상품 4개 표시
  - 클래식 코튼 티셔츠 (₩35,000)
  - 프리미엄 패딩 재킷 (₩89,000)
  - 레인코트 (₩45,000)
  - 리본 스카프 (₩15,000)
- [ ] 상품 카드 hover 효과 (확대)
- [ ] 뱃지 표시 (신상품, 베스트셀러)
- [ ] 색상 옵션 미리보기 (원형 색상 점)
- [ ] 반응형: 모바일(1열) → 데스크톱(4열)

**테스트 시나리오**:
1. **카테고리 필터 클릭**
   - "이너웨어" 클릭 → 티셔츠만 표시되는지 확인
   - "아우터" 클릭 → 패딩, 레인코트만 표시되는지 확인
   - "전체" 클릭 → 다시 4개 모두 표시되는지 확인

2. **상품 카드 클릭**
   - 아무 상품이나 클릭 → 상세 페이지로 이동하는지 확인

**스크린샷 찍어주세요**: 
- 전체 상품 리스트
- 카테고리 필터 적용된 화면

---

### 3️⃣ 상품 상세 페이지 (`/products/[slug]`)

**예시**: http://localhost:3000/products/classic-cotton-tshirt

**확인 사항**:
- [ ] 이미지 갤러리
  - 메인 이미지 표시
  - 썸네일 이미지 (여러 개면)
  - 썸네일 클릭 시 메인 이미지 변경
- [ ] 상품 정보
  - 카테고리 뱃지
  - 상품명
  - 가격
  - 상품 설명
- [ ] 사이즈 선택 버튼 (S, M, L, XL)
- [ ] 색상 선택 버튼 (White, Black, Navy)
- [ ] 수량 조절 (+/-)
- [ ] 재고 상태 표시
- [ ] 장바구니 담기 버튼
- [ ] 바로 구매 버튼
- [ ] 배송/반품 정보 카드
- [ ] 관련 상품 추천 (같은 카테고리)

**테스트 시나리오 1: 옵션 미선택 에러**
1. 사이즈/색상 선택하지 않고 "장바구니 담기" 클릭
2. **예상 결과**: "사이즈를 선택해주세요" alert 표시

**테스트 시나리오 2: 장바구니 담기**
1. 사이즈 선택 (예: M)
2. 색상 선택 (예: Black)
3. 수량 2개로 조절
4. "장바구니 담기" 클릭
5. **예상 결과**: 
   - "장바구니에 추가되었습니다" alert
   - Header의 장바구니 아이콘에 빨간 숫자 "2" 표시

**테스트 시나리오 3: 바로 구매**
1. 사이즈/색상 선택
2. "바로 구매" 버튼 클릭
3. **예상 결과**: 장바구니 페이지로 자동 이동

**스크린샷 찍어주세요**:
- 상품 상세 페이지 전체
- 옵션 선택 전/후
- Header에 장바구니 숫자 표시된 모습

---

### 4️⃣ 장바구니 페이지 (`/cart`)

**확인 사항**:
- [ ] 장바구니 아이템 목록
  - 상품 이미지
  - 상품명
  - 옵션 (사이즈, 색상)
  - 수량 조절 버튼 (+/-)
  - 개별 가격 및 합계
  - 삭제 버튼 (X)
- [ ] 주문 요약 카드 (우측, sticky)
  - 상품 금액
  - 배송비
  - 총 결제 금액
- [ ] "전체 삭제" 버튼
- [ ] "주문하기" 버튼
- [ ] "쇼핑 계속하기" 버튼

**테스트 시나리오 1: 수량 조절**
1. 장바구니에 상품 있는 상태에서
2. + 버튼 클릭 → 수량 증가, 금액 자동 업데이트
3. - 버튼 클릭 → 수량 감소, 금액 자동 업데이트
4. **예상 결과**: 주문 요약의 금액도 실시간 업데이트

**테스트 시나리오 2: 배송비 계산**
1. 장바구니에 30,000원 상품만 담기
2. **예상 결과**: 배송비 3,000원 표시
3. 상품 추가해서 50,000원 이상 만들기
4. **예상 결과**: 배송비 "무료" 표시

**테스트 시나리오 3: 아이템 삭제**
1. X 버튼 클릭
2. **예상 결과**: 해당 아이템 즉시 삭제
3. Header의 장바구니 숫자도 감소

**테스트 시나리오 4: 빈 장바구니**
1. "전체 삭제" 클릭
2. **예상 결과**: 
   - "장바구니가 비어있습니다" 메시지
   - "상품 둘러보기" 버튼 표시

**스크린샷 찍어주세요**:
- 장바구니에 상품 여러 개 담긴 상태
- 배송비 무료/유료 각각
- 빈 장바구니 상태

## 🔄 전체 플로우 테스트

### 시나리오: 상품 구매 전체 과정

1. **홈페이지** 접속
2. Header "판매상품" 클릭 → 리스트 페이지 이동

5. "장바구니 담기" 클릭
6. 다른 상품도 추가 (예: 리본 스카프)
7. Header 장바구니 아이콘 확인 (숫자 증가)
8. 장바구니 아이콘 클릭 → 장바구니 페이지 이동
9. 수량 조절 및 확인
10. 주문 요약 금액 확인
