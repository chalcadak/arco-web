## ✅ 새로 추가된 테스트 항목

### 5️⃣ 촬영룩 리스트 페이지 (`/photoshoots`)

**URL**: http://localhost:3000/photoshoots

**확인 사항**:
- [ ] Hero Section: "촬영룩" 타이틀
- [ ] 카테고리 필터 버튼 4개 (전체, 에디토리얼, 시즌 스페셜, 특별한 날)
- [ ] 촬영룩 3개 표시
  - 빈티지 에디토리얼 (₩150,000 / 90분)
  - 봄날의 피크닉 (₩120,000 / 60분)
  - 생일파티 스페셜 (₩100,000 / 60분)
- [ ] 촬영 시간 배지 (우측 하단)
- [ ] 카드 hover 효과
- [ ] 반응형: 모바일(1열) → 데스크톱(4열)

**테스트 시나리오**:
1. **카테고리 필터 클릭**
   - "에디토리얼" 클릭 → 빈티지 에디토리얼만 표시
   - "시즌 스페셜" 클릭 → 봄날의 피크닉만 표시
   - "전체" 클릭 → 다시 3개 모두 표시

2. **촬영룩 카드 클릭**
   - 아무 촬영룩이나 클릭 → 상세 페이지로 이동

**스크린샷 찍어주세요**:
- 전체 촬영룩 리스트
- 카테고리 필터 적용된 화면

- [ ] 예약하기 버튼
- [ ] 전화 문의 버튼
- [ ] 촬영 정보 카드
  - 촬영 시간
  - 예약 가능
  - 납품 기간
  - 취소/변경

2. **전화 문의 버튼 클릭**
   - **예상 결과**: 전화 걸기 앱 실행 (모바일) 또는 전화번호 표시

## 🔄 전체 플로우 테스트 (업데이트)

### 시나리오 1: 판매상품 구매 플로우
1. 홈페이지 → "판매상품" 클릭
2. 상품 리스트 → 원하는 상품 클릭
3. 상품 상세 → 사이즈/색상 선택 → "장바구니 담기"
4. Header 장바구니 숫자 증가 확인
5. 장바구니 아이콘 클릭 → 장바구니 페이지
6. 수량 조절 및 확인

### 시나리오 2: 촬영룩 예약 플로우 (NEW!)
1. 홈페이지 → "촬영룩" 클릭
2. 촬영룩 리스트 → 원하는 촬영룩 클릭
3. 촬영룩 상세 → 포함 항목 및 요구사항 확인
4. "예약하기" 버튼 클릭 (현재는 alert만 표시)
5. (향후: 예약 폼 → 날짜/시간/반려견 정보 입력)

## 🎨 판매상품 vs 촬영룩 비교

| 항목 | 판매상품 | 촬영룩 |
|------|----------|---------|
| **가격** | 상품 가격 | 촬영 서비스 가격 |
| **옵션** | 사이즈, 색상 | (없음) |
| **수량** | 수량 선택 | (없음) |
| **재고** | 재고 수량 표시 | (없음) |
| **액션** | 장바구니 담기, 바로 구매 | 예약하기, 전화 문의 |
| **추가 정보** | 배송/반품 정보 | 촬영 시간, 포함 항목, 요구사항 |
| **배지** | 신상품, 베스트셀러 | 신규, 인기 |
| **시간 표시** | (없음) | 촬영 시간 (분) |
